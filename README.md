# dat-ls

Small program that lists all the changes in a dat

```
npm install -g dat-ls
```

## Usage

```
dat-ls <dat-link>
```

Here is an example

```
> dat-ls 8a1b6864f290a1204a3fe048ced34865a24f5fdef4ce3753bab613f66636587d
```
